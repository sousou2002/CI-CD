# CI CD - Ejercicios

Repositorio de ejemplo con archivos iniciales para practicar CI/CD.

## Requisitos
- Python 3.8+

## Instalación
Instala dependencias:

```powershell
pip install -r requirements.txt
```

## Ejecutar
Para probar el script principal:

```powershell
python starter_files/hello.py
```

## Tests
Ejecutar tests con `pytest`:

```powershell
pytest
```

## Notas
Se agregó `.gitignore` para ignorar archivos compilados, entornos virtuales y archivos de IDE/OS.
